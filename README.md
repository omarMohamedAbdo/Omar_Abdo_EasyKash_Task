### Installation :

  - Install NodeJS and NPM from https://nodejs.org/en/download/.

  - Download or clone the tutorial project source code from : https://github.com/omarMohamedAbdo/Omar_Abdo_EasyKash_Task.

## To run Server :

```sh
$ cd Server
$ npm install
$ npm start
```

## To run Client :

```sh
$ cd Client
$ npm install
$ ng serve
```
## To run tests :

```sh
$ cd Server
$ npm test
```
